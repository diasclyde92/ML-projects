
# Machine Learning Playground

An interactive Streamlit dashboard showcasing multiple machine learning models.

## Live Demo
https://ml-projects-e4tfgytwzxb5dz8ecwggmy.streamlit.app/

## Projects Included

### House Price Predictor
Predict house prices based on square footage and number of bedrooms.

### Spam Email Classifier
Classifies whether an email message is spam or not using NLP techniques.

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- Streamlit
- Joblib

This repository contains multiple end-to-end machine learning demo projects.

Projects included:

1. House Price Predictor
2. Spam Email Classifier

A unified Streamlit dashboard allows testing both models.

Run locally:

pip install -r requirements.txt
streamlit run ml-dashboard/streamlit_app.py
